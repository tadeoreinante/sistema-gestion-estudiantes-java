# Sistema de Turnos - Java

Proyecto de practica realizado en Java utilizando Programación Orientada a Objetos.

## Funcionalidades
- Gestión de pacientes
- Creación de turnos
- Organización de información
- Uso de clases y objetos

## Tecnologías utilizadas
- Java
- BlueJ
- Git/GitHub

## Objetivo
Proyecto académico orientado a la práctica de lógica de programación y Programación Orientada a Objetos.
